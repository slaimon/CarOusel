Documentation

the track is described by its left and right curbs, when a curb is a sequence of n 3d points describing a closed line (so point n-1 is connected to point 0 ). The two curbs have the same number of points and can be draw with a GL_TRIANGLE_STRIP providing vertices as curb[0][0],curb[1][0],curb[0][1],curb[1][1] and so on...
